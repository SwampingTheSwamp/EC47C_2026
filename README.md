## Descrição

Projeto desenvolvido para a disciplina de Programação Web Front-End da Universidade Tecnológica Federal do Paraná (UTFPR).
É uma aplicação web voltada para a área da educação. A proposta é criar um espaço digital para facilitar o acesso de estudantes a conteúdos de apoio, materiais complementares, dicas de estudo e informações sobre oportunidades educacionais.

A ideia do projeto é contribuir para a democratização do acesso à informação e incentivar o desenvolvimento educacional da comunidade.

## Objetivo

Desenvolver uma interface web simples, intuitiva e acessível, aplicando os conceitos de HTML e CSS estudados em sala de aula.

## Público-alvo

- Estudantes do ensino médio e superior
- Pessoas em busca de materiais de apoio para estudos
- Comunidade local interessada em oportunidades educacionais

## Relação com os Objetivos de Desenvolvimento Sustentável (ODS)

Este projeto está alinhado com o **ODS 4 – Educação de Qualidade**, que busca assegurar educação inclusiva, equitativa e de qualidade, além de promover oportunidades de aprendizagem ao longo da vida.

## Funcionalidades

- Página de cadastro de usuários
- Página de login
- Página principal com conteúdos e informações educacionais
- Navegação entre páginas
- Layout responsivo e organizado
